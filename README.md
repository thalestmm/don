# don

Simple, CLI-based, personal finances portfolio manager.

Data is stored locally within a `json` ledger file for ease of portability and usage within other API's.

You can have multiple ledger files, each with a unique name and currency.

**Project under active development, expect breaking API changes.**

## Installation

```bash
go install github.com/thalestmm/don@latest
```
